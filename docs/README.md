## Documentation Philosophy

This documentation follows the [Diátaxis framework](https://diataxis.fr/):

- **Tutorials** (learning-oriented) - Step-by-step lessons for beginners
- **How-to Guides** (problem-oriented) - Practical solutions to specific problems
- **Technical Reference** (information-oriented) - Precise API descriptions
- **Explanation** (understanding-oriented) - Clarification and discussion
