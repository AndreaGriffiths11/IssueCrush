üìö Usage Instructions

1 Installation

The component requires framer-motion (already installed):

npm install framer-motion
2 Import the Component
import CraftCard from './src/components/CraftCard';
import './src/components/CraftCard.css';
3 Basic Usage
<CraftCard delay={0.2}>
  <h2>My Card Title</h2>
  <p>Card content goes here...</p>
</CraftCard>
4 Grid with Stagger
import { CardGrid } from './src/components';

<CardGrid
  data={myData}
  staggerDelay={0.1}
  renderCard={(item) => (
    <h3>{item.title}</h3>
  )}
/>
5 View Examples

Check out the comprehensive examples in:

  • src/components/CraftCardDemo.tsx - Full demo with multiple examples
  • src/components/README.md - Complete documentation
6 Run the Web App

To see the component in action with the IssueCrush app:

npm run web-dev

Then open your browser to test the interactive animations.
